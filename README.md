# OS Lab Project

## 🛠️ Setup Guide (Using VS Code)

### 1. Install Required Tools
- Install **C compiler (Clang/GCC)**  
- In VS Code, install:
  - **C/C++ Extension Pack**
  - **Code Runner** (by Jun Han)

---

### 2. Navigate to the File
Open terminal in VS Code and `cd` into the correct folder:
```
cd Lab-x
```

---

### 3. Compile and Run C Program
Compile:
```
gcc filename.c -o filename
```

Run (on mac):
```
./filename
```
Run (on windows):
```
filename
```

